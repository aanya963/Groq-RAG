Project flow is:

    PDF
        ↓
    Extract text
        ↓
    Chunk text
        ↓
    Create embeddings
        ↓
    Store embeddings in Vector Database

Then during chat:

    User Question
        ↓
    Create embedding
        ↓
    Search similar chunks in vector DB
        ↓
    Send chunks as context to LLM
        ↓
    LLM generates answer
